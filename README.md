# **Blockchain Demo (Flask)**

This is a simple blockchain simulation project using Python Flask.

## **Features**

• Live hash update  
• Block mining  
• Chain validation  
• Interactive UI  

## **Tech Stack**

• Python (Flask)  
• HTML, CSS, JavaScript  

## **How to Run**

1. Install Flask  
2. Run `python app.py`  
3. Open http://127.0.0.1:5000/
